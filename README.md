# 🏨 BookMyStayApp

A **console-based Java application** that simulates an online hotel booking system.  
The app allows users to search, book, and manage hotel stays while demonstrating core **Java, OOP, and system design concepts**.

---

## 📌 Overview

The application models:

- 🏨 Hotels and room types  
- 👤 User booking operations  
- 📅 Reservation management  
- 💳 Payment handling  
- ⚠️ Exception handling for booking validation  

The system evolves step-by-step to reflect **real-world booking workflows**.

---

## 🎯 Objectives

- Simulate a hotel booking platform  
- Apply Object-Oriented Programming concepts  
- Implement search, sort, and booking logic  
- Handle invalid booking scenarios  
- Build structured and scalable code  

---

## 🧩 Core Functionalities

### 🔹 Hotel Management
- Add and manage hotel details  
- Store location, rating, and room availability  

### 🔹 Room Types
- Standard Room  
- Deluxe Room  
- Suite  

---

### 🔹 Booking System
- Create bookings for users  
- Check room availability  
- Assign rooms dynamically  

### 🔹 Reservation Tracking
- Store booking ID, user details, and duration  
- Maintain booking history  

---

### 🔹 Search Functionality
- Search hotels by:
  - Name  
  - Location  
  - Availability  

---

### 🔹 Sorting Features
- Sort hotels by rating or price  
- Uses both manual and built-in sorting approaches  

---

### 🔹 Payment Handling
- Simulates booking payment process  
- Validates payment completion  

---

### 🔹 Exception Handling
- Prevent booking when rooms are unavailable  
- Handle invalid inputs gracefully  
- Ensure system stability using try-catch
